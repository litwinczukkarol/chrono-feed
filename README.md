## 🔖 Instrukcja Bookmarkleta (v0.1)

Aby szybko przesyłać listę seriali z Serializd do Chrono-Feed:

1. Stwórz nową zakładkę w przeglądarce i nazwij ją **Pobierz do Chrono-Feed**.
2. W polu **Adres URL** wklej poniższy kod:

```javascript
